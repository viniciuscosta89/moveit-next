# NLW#4 with React/NextJS
"Move.it" is a Pomodoro Timer app with some challenges to fulfill.
It was made @ [Rocketseat's NLW#4](https://rocketseat.com.br).

You can try it [here](https://moveit-next-tau.vercel.app).

## Technologies
- React
- Next.js
- NextAuth.js
- Typescript
- styled-components
- Framer Motion
- js-cookie
